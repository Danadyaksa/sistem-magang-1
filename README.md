# Sistem Informasi Magang DISDIKPORA

Sistem Informasi Magang DISDIKPORA adalah aplikasi berbasis web yang dirancang untuk membantu proses pengelolaan kegiatan magang pada Dinas Pendidikan, Kepemudaan, dan Olahraga (DISDIKPORA).  
Aplikasi ini memfasilitasi pendaftaran magang oleh mahasiswa serta pengelolaan data magang oleh admin secara terstruktur dan terkomputerisasi.

Project ini dikembangkan sebagai bagian dari kegiatan **Kerja Praktik (KP)** / pengembangan sistem informasi.

---

## Tujuan Sistem
- Mempermudah proses pendaftaran magang secara online
- Mengelola data peserta magang secara terpusat
- Mengurangi proses manual dan penggunaan dokumen fisik
- Menyediakan sistem yang rapi, terstruktur, dan mudah dikembangkan

---

## Teknologi yang Digunakan

- **Next.js** – Framework React berbasis server & client
- **TypeScript** – Bahasa pemrograman dengan type safety
- **Tailwind CSS** – Styling berbasis utility-first
- **Prisma ORM** – Pengelolaan database
- **shadcn/ui**
- **Node.js**
- **Database**: Supabase
